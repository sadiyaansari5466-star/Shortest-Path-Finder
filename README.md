# Pathfinding Project (BFS & Dijkstra)

## About the Project
This project is about finding the shortest path between nodes using BFS and Dijkstra’s algorithm. I built it to understand how real-world systems like maps and navigation tools work.

## Where it is used
- Navigation systems like Google Maps  
- Network routing  
- Game movement logic  

## What I used
- Graph representation using Python dictionaries  
- BFS (Breadth First Search)  
- Dijkstra’s Algorithm  
- Queue and Priority Queue  

## Sample Output

BFS Traversal:
A B C D E F  

Dijkstra Shortest Path:
A -> B = 1  
A -> F = 5  

## What I learned
While working on this, I got a better understanding of how shortest path algorithms work and how efficiently they can solve real problems.
